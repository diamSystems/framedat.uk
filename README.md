# framed@ | framedat.uk

Static preview site for **framed@ Limited** (Co. No. 16987092) — a Northern arts and music platform.

**Live**: [adam-scholey.github.io/framedat.uk](https://adam-scholey.github.io/framedat.uk)

## Pages

- **Home** — Brand statement, upcoming event hero, email sign-up CTA
- **Events** — Upcoming shows across Manchester, Leeds, Liverpool
- **Archive** — Photo gallery and video documentation
- **About** — The mission, values, and team
- **Contact** — Get in touch form and direct email

## Design

- Pure black (#000) on white (#FFF)
- Bebas Neue display type + Inter body text
- No gradients, no clutter — bold and minimal
- Fully responsive (mobile, tablet, desktop)

## Stack

HTML5, CSS3, vanilla JavaScript. No frameworks, no dependencies.
